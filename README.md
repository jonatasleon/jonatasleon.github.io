Jonatas Leon's Blog
===

This is my personal blog, where I write about my experiences and thoughts on software development. 
It it built using [Jekyll][jekyllrb] and the [Chirpy][chirpy] theme.

The content is written in Brazilian Portuguese and focused on the Brazilian software development community.

[jekyllrb]: https://jekyllrb.com/
[chirpy]: https://github.com/cotes2020/jekyll-theme-chirpy
